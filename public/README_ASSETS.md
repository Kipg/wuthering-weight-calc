# Public Assets

请将游戏资源放置在以下目录：

## 目录结构

```
public/
├── characters/        # 角色立绘图片
│   └── carlotta.png
├── weapons/          # 武器图标
│   └── unshakable-crown.png
├── element_icon/     # 元素图标
│   ├── aero.png      # 气动
│   ├── fusion.png    # 热熔
│   ├── spectro.png   # 衍射
│   ├── glacio.png    # 冷凝
│   ├── havoc.png     # 湮灭
│   └── electro.png   # 导电
└── kuku.gif         # Logo
```

## 图片规格建议

- **角色立绘**: 推荐 512x512 或更高，PNG 格式，透明背景
- **武器图标**: 推荐 256x256，PNG 格式，透明背景
- **元素图标**: 推荐 128x128，PNG 格式，透明背景
- **Logo**: 推荐 400x80，GIF 或 PNG 格式

---

**注意**: 本项目不提供游戏资源文件，请自行准备或使用占位图片。
